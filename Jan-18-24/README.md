# Project Title: FlutterFlix - Static Clone

### Explore FlutterFlix, a static Netflix-inspired clone featuring a curated selection of great movies and web series. Navigate seamlessly through sections like Movies, Web Series, and Most Popular using the intuitive bottom navigation bar with options for Home, Search, Downloads, and Play. Immerse yourself in the world of entertainment with this sleek and user-friendly Flutter project.
